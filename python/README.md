# Modular Analysis Tool

A command-line analysis tool showcasing modular architecture for AI agent automation.

## Features

- Modular architecture with automatic module discovery
- Easy module addition/removal (folder + registry entry)
- Standardized module contracts
- Built-in data analysis modules

## Usage

```bash
poetry run python -m modular_analysis
```
